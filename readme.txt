このプログラムは7z.exeがカレントディレクトリ または 環境変数PATHに登録されたディレクトリに存在することを前提に作られています

[利点]
インストールするプログラムのDLLファイルの重複を無くしサイズを節約
ドラックアンドドロップだけでインストールができる

<インストールモード(Mode0)>
1.	.7ziファイルをProgramManagerにドラックアンドドロップ
2.	インストール&ProgramManagerと同じフォルダにショートカット(.ypr)が生成される


<呼び出しモード(Mode1)>
1.	.yprファイルをProgramManagerにドラックアンドドロップ
2.	呼び出しプログラムにドラックアンドドロップしたいものをウィンドウにドラックアンドドロップしてEnter ない場合は何も書かずにEnter

<管理モード(Mode2)>
ProgramManager単体で起動(ドラックアンドドロップ無し)すると、管理画面が開く
アンインストール パスの取得 ショートカットを作成 ができる
